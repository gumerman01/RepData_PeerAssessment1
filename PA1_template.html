Imputing missing values

Calculate and report the total number of missing values in the dataset.

sum(is.na(activity$steps))
## [1] 2304

Devise a strategy for filling in all of the missing values in the dataset. The strategy does not need to be sophisticated. Use the mean for that date. Create a new dataset that is equal to the original dataset but with the missing data filled in.

actWoNA <- activity
meanDate <- aggregate(actWoNA$steps, list(date = actWoNA$date), mean, na.rm=TRUE)
names(meanDate) <- c("date","steps")
for (i in 1:length(actWoNA$steps)) {
    if (is.na(actWoNA$steps[i])) {
        dateNA <- actWoNA$date[i]
        dateMean <- filter(meanDate,date %in% dateNA)
        if (is.nan(dateMean$steps)) {
            dateMean$steps <- 0
        }
        actWoNA$steps[i] <- dateMean$steps
    }
}
